# Bazil Facebook Ads — Quick Setup Checklist

Use this as a step-by-step when you're ready to create the campaign in Ads Manager.

## Before You Start

- [ ] Facebook Business Page for Bazil exists (or create one)
- [ ] Ad account is set up with a verified payment method
- [ ] Payment card is set as **Default** (not Backup) — this is critical for delivery
- [ ] You're logged into facebook.com / business.facebook.com

## Campaign Level

- [ ] Go to adsmanager.facebook.com → click green **+ Create**
- [ ] Objective: **Traffic**
- [ ] Campaign name: `Bazil — Meal Planning Traffic`
- [ ] Campaign type: **Manual** (not Advantage+ recommended)
- [ ] Special categories: None
- [ ] Click Continue

## Ad Set Level

- [ ] Ad set name: `US Health-Conscious 18-45`
- [ ] Performance goal: **Landing page views**
- [ ] Daily budget: **£10.00**
- [ ] Schedule: Start today, no end date
- [ ] Audience: Advantage+ with these interests:
  - Meal prep / Meal planning
  - Healthy eating
  - Physical fitness
  - Weight loss
  - Cooking
- [ ] Age: 18–45
- [ ] Location: United States
- [ ] Language: English
- [ ] Placements: Advantage+ (automatic)

## Ad Level (repeat for each of the 4 ads)

- [ ] Ad name: `Ad 1 — Pain Point` (or Ad 2, 3, 4)
- [ ] Identity: Select your Bazil Facebook Page
- [ ] Format: Single image
- [ ] Upload image (see 03_IMAGE_GUIDE.md for which image per ad)
- [ ] Crop to square (1:1) for best cross-placement performance
- [ ] Primary text: Copy from 02_AD_COPY_VARIATIONS.md
- [ ] Headline: Copy from 02_AD_COPY_VARIATIONS.md
- [ ] Description: Copy from 02_AD_COPY_VARIATIONS.md
- [ ] CTA: **Learn More**
- [ ] Website URL: `https://www.bazilcooking.com`
- [ ] Enhancements: Leave off for now

## Publish

- [ ] Review all 4 ads in the preview panel
- [ ] Check mobile preview specifically (most traffic will be mobile)
- [ ] Click **Publish**
- [ ] Ads enter review (1–24 hours typically)

## Post-Launch (Days 1–7)

- [ ] Days 1–3: DON'T TOUCH ANYTHING — learning phase
- [ ] Day 4: Check CTR and CPC for each ad
- [ ] Kill any ad with CTR below 0.8%
- [ ] Day 7: Identify the winner, consider pausing the rest
- [ ] Week 2+: Scale winner to £15–20/day if performing well
